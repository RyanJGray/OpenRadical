Detailed Description

template<class _Tp, class _Alloc, bool __is_static>
class std::_Deque_alloc_base< _Tp, _Alloc, __is_static >

@maint Primary default version. @endmaint

@maint Deque base class. It has two purposes. First, its constructor and destructor allocate (but don't initialize) storage. This makes exception safety easier. Second, the base class encapsulates all of the differences between SGI-style allocators and standard-conforming allocators. There are two versions: this ordinary one, and the space-saving specialization for instanceless allocators. @endmaint
