Detailed Description

template<class _Tp, class _Alloc>
class std::_Deque_base< _Tp, _Alloc >

@maint Deque base class. Using _Alloc_traits in the instantiation of the parent class provides the compile-time dispatching mentioned in the parent's docs. This class provides the unified face for deque's allocation.

Nothing in this class ever constructs or destroys an actual Tp element. (Deque handles that itself.) Only/All memory management is performed here. @endmaint
