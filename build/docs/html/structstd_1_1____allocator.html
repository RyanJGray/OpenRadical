Detailed Description

template<class _Tp, class _Alloc>
struct std::__allocator< _Tp, _Alloc >

@maint Allocator adaptor to turn an "SGI" style allocator (e.g., __alloc, __malloc_alloc_template) into a "standard" conforming allocator. Note that this adaptor does not assume that all objects of the underlying alloc class are identical, nor does it assume that all of the underlying alloc's member functions are static member functions. Note, also, that __allocator<_Tp, __alloc> is essentially the same thing as allocator<_Tp>. @endmaint (See allocators info for more.)
